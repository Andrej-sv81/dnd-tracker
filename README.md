# dnd-tracker
DnD Dungeon Master Player Tracker (DevOps-Focused Full-Stack Project)

This project is a DevOps-first full-stack application designed to simulate a production-grade system while implementing a simple Dungeons & Dragons player tracker.
The primary goal is learning real-world DevOps practices, not just building features.

Tech stack:
  * Backend: Go (Gin framework)
  * Frontend: Angular
  * Database: PostgreSQL
  * Reverse Proxy: nginx
  * Containerization: Docker
  * CI/CD: GitHub Actions
  * Infrastructure: Terraform
  * Cloud: Oracle Cloud
